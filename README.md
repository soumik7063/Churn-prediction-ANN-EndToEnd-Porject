# 📊 Customer Churn Prediction App (Streamlit + TensorFlow)

A machine learning web application that predicts the probability of customer churn using a trained **TensorFlow/Keras model**. The app is built with **Streamlit** and uses preprocessing pipelines including **Label Encoding, One-Hot Encoding, and Feature Scaling**.

---

## 🚀 Features

- Interactive UI using Streamlit  
- Real-time churn prediction  
- Preprocessing with encoders and scaler  
- Supports categorical + numerical inputs  
- Displays churn probability instantly  

---

## 🛠️ Tech Stack

- Python  
- TensorFlow / Keras  
- Streamlit  
- Pandas  
- Scikit-learn
